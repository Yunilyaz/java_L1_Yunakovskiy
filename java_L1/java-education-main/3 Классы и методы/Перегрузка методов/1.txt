В цикле for (1) каждый элемент массива имеет тип Number, что заставляет компилятор выбрать метод 
с параметром типа Number, независимо от фактического типа каждого элемента

В вызовах (2) аргументы передаются с конкретными типами (Integer, int, Float, double), 
что позволяет компилятору выбрать наиболее подходящую перегруженную версию метода, соответствующую типу переданного аргумента
