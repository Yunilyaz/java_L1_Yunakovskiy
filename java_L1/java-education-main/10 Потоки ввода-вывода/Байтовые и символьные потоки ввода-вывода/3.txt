InputStream/OutputStream предназначены для работы с байтовыми данными. 
Эти классы отлично подходят для работы с файлами, сетевыми сокетами и любыми другими данными, которые представляют собой потоки байтов
Reader/Writer предназначены для работы с символами и текстом. 
Эти классы учитывают кодировку символов и упрощают работу с текстовыми данными
Методы классов Reader и Writer более интуитивны для работы с текстом, 
так как они автоматически обрабатывают преобразование между символами и байтами
