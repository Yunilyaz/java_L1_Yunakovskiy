Существует 4 основный класса исключений:Throwable, Error, Exception, RuntimeException
Их назначения:
Throwable - базовый класс для всех ошибок и исключений
Error - используется для представления серьёзных ошибок (переполнение стека или нехватка памяти)
Exception - базовый класс для всех исключений, которые могут быть выброшены при работе программы
RuntimeException - подкласс Exception, включающий исключения, возникающие при выполнении программы (обращение к несуществующему индуксу массива)
