//Практика 1
Можно ли использовать конструкцию (try без catch и finally)?
Нет, нельзя - в Java нельзя использовать блок try без хотя бы одного блока catch или finally, иначе - ощибка компиляции


//Практика 2
Можно ли использовать конструкцию (try только с finally)?
Да, можно - это позволяет выполнять код в блоке finally, независимо от того, произошло ли исключение в блоке try 
Однако исключение не будет обработано, если оно произойдет в блоке try

//Практика 3
Можно ли использовать конструкцию (try с двумя finally)?
Нет, нельзя - в Java нельзя использовать два блока finally для одного блока try
Каждый блок try может иметь только один соответствующий блок finally, иначе - ошибка компиляции
