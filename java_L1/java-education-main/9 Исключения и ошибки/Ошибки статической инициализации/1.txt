Программа выдаст сообщение об ошибке: " Exception in thread "main" java.lang.NumberFormatException: For input string: "Y-" "

Метод Integer.parseInt(String s) пытается преобразовать строку в целое число, хотя строка "Y-" не является корректным представлением целого числа
В результате выполнения Integer.parseInt("Y-") будет выброшено исключение NumberFormatException, показывающее, что строка не может быть преобразована в число
Поскольку исключение возникает во время статической инициализации, выполнение программы прерывается, и метод main не будет вызван.
